# 📚 Exercícios de JavaScript

50 exercícios de JavaScript organizados por tema, do básico ao avançado.

---

## 🔰 01 — Fundamentos (1–10)

**1.** Crie um programa que receba dois números e retorne: soma, subtração, multiplicação e divisão.

**2.** Crie um programa que receba um número e retorne seu dobro e seu triplo.

**3.** Crie um programa que receba um número e retorne seu valor absoluto.

**4.** Crie um programa que receba dois números e retorne o maior deles.

**5.** Crie um programa que receba três números e retorne a média.

**6.** Crie um programa que receba um número e verifique se é positivo, negativo ou zero.

**7.** Crie um programa que receba um número e verifique se é par ou ímpar.

**8.** Crie um programa que receba dois números e verifique se ambos são maiores que 10.

**9.** Crie um programa que receba um número e verifique se está entre 20 e 100.

**10.** Crie um programa que receba uma idade e classifique: criança, adolescente, adulto ou idoso.

---

## 🔀 02 — Condicionais (11–15)

**11.** Receba uma nota (0–10) e retorne "aprovado" ou "reprovado".

**12.** Receba duas notas, calcule a média e classifique o resultado.

**13.** Receba um número e verifique se é múltiplo de 3 e de 5.

**14.** Receba um turno (M, T, N) e retorne a saudação correta.

**15.** Receba um salário e aplique aumento baseado em faixas salariais.

---

## 🔁 03 — Loops (16–25)

**16.** Imprima números de 1 até N.

**17.** Some números de 1 até N.

**18.** Mostre a tabuada de um número.

**19.** Conte quantos números pares existem até N.

**20.** Imprima números de N até 1.

**21.** Calcule o fatorial de um número.

**22.** Gere a sequência de Fibonacci.

**23.** Leia números até receber 0 e retorne a soma total.

**24.** Verifique se um número é primo.

**25.** Crie uma pirâmide numérica.

---

## 🧩 04 — Funções (26–30)

**26.** Crie uma função que soma dois números.

**27.** Crie uma arrow function que retorna o dobro de um número.

**28.** Crie uma função que verifica se um número é par.

**29.** Crie uma função que recebe um array e retorna a soma dos elementos.

**30.** Crie uma função que recebe dois números e retorna o maior.

---

## 📦 05 — Arrays Base (31–35)

**31.** Percorra um array e imprima todos os valores.

**32.** Conte quantos elementos existem sem usar `.length`.

**33.** Encontre o maior valor do array.

**34.** Some todos os valores do array.

**35.** Crie um novo array apenas com números pares (sem usar `.filter`).

---

## ⚙️ 06 — Métodos de Array (36–45)

**36.** Use `.map()` para dobrar todos os valores.

**37.** Use `.map()` para transformar nomes em maiúsculo.

**38.** Use `.filter()` para retornar números maiores que 50.

**39.** Use `.filter()` para retornar apenas números pares.

**40.** Use `.find()` para encontrar um número maior que 100.

**41.** Use `.find()` para buscar um objeto por id.

**42.** Use `.includes()` para verificar se o array contém um valor.

**43.** Use `.some()` para verificar se existe algum número negativo.

**44.** Use `.every()` para verificar se todos os valores são positivos.

**45.** Use `.map()` + `.filter()` para retornar os nomes de produtos caros.

---

## 🧠 07 — Desafios (46–50)

**46.** Crie um sistema simples de login usando um array de usuários.

**47.** Crie um carrinho de compras e calcule o total com `.reduce()`.

**48.** Implemente uma busca linear.

**49.** Implemente uma busca binária em um array ordenado.

**50.** Simule uma fila de atendimento (estrutura queue).

---

## 🚀 Como usar

Resolva cada exercício na pasta correspondente. Crie um arquivo `.js` por exercício, por exemplo:

```
01-fundamentos/
├── ex01.js
├── ex02.js
└── ...
```

Execute suas soluções com Node.js:

```bash
node 01-fundamentos/ex01.js
```

---

> 💡 **Dica:** Tente resolver cada exercício por conta própria antes de buscar referências. O objetivo é construir o raciocínio lógico passo a passo.
